# Hand Tracking & Mouse Control

สคริปต์ Python ใช้ไลบรารี Mediapipe และ OpenCV ดึงภาพจากบริการ Camera Streamer เพื่อแปลงการเคลื่อนไหวของมือเป็นการควบคุมเมาส์บนระบบปฏิบัติการ

## รูปแบบการรันโปรแกรม

1. **โหมดทดสอบในเครื่องคอมพิวเตอร์ (Software Mouse):**
   ควบคุมเมาส์บนระบบปฏิบัติการของคอมพิวเตอร์ที่รันสคริปต์นี้โดยตรง (ใช้ PyAutoGUI)
   ```bash
   python hand/main.py
   ```

2. **โหมดส่งคำสั่งผ่านบลูทูธทางอ้อม (API Mouse - แนะนำ):**
   ส่งคำสั่งพิกัดเมาส์ไปยังเซิร์ฟเวอร์บลูทูธผ่าน HTTP POST เพื่อให้ส่งต่อบอร์ด ESP32-C3 อีกที (สามารถเปิดทิ้งไว้พร้อมกับเซิร์ฟเวอร์หลักและ Next.js ได้โดยไม่แย่งพอร์ตพอร์ต)
   ```bash
   python hand/main.py api
   ```

3. **โหมดต่อสายบอร์ดโดยตรง (BLE Mouse):**
   เชื่อมต่อพอร์ต Serial ตรงไปยังบอร์ด ESP32-C3 (ต้องปิดตัวเซิร์ฟเวอร์ `main.py` ฝั่งบลูทูธก่อนเพื่อไม่ให้เกิดการชนกันของพอร์ต)
   ```bash
   python hand/main.py ble COM19
   ```

## ข้อมูลการทำงาน
- ดึงภาพสตรีมวิดีโอจาก `http://localhost:5000/api/video_feed` แทนการต่อตรงกับพอร์ตกล้อง
- ปิดระบบ FAILSAFE ของ PyAutoGUI เพื่อป้องกันไม่ให้โปรแกรมแครชเมื่อเมาส์เข้าใกล้มุมจอ
