### [Manual] hand-detection (ไม่มีพอร์ตรับสายเข้า)
บริการรันโปรแกรม MediaPipe เพื่อตรวจจับท่าทางมือและจำลองเมาส์ โดยการดึงภาพจากพอร์ต 5000 มาประมวลผลต่อ และส่งคำสั่งควบคุมทีวี/เมาส์ผ่าน API

**วิธีการรัน:**
- รันด้วยพารามิเตอร์ `api` เพื่อเข้าสู่ Async API Mode ในการควบคุม
- คำสั่งเบื้องหลัง: `python main.py api`
