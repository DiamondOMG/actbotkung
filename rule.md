# กฏในการทำโปรเจค และ สื่อสารกับ Dev

1. **การใช้งาน Shell**: ใช้ Git Bash เป็นหลัก
2. **การตั้งชื่อ (Naming Convention)**:
   - ใช้เครื่องหมายขีดกลาง (`-`) กับชื่อโฟลเดอร์ และไฟล์ asset ต่างๆ
   - ใช้เครื่องหมายขีดล่าง (`_`) กับชื่อไฟล์สคริปต์/โค้ด และตัวแปรทั้งหมด (เช่น `main_py`, `software_mouse_py`)
3. **การรันคำสั่ง Terminal**: ห้ามให้ AI รัน Terminal เองเด็ดขาด มีหน้าที่แนะนำและเขียนคำสั่งให้ User คัดลอกไปรันเอง แล้วนำผลลัพธ์มาสนทนาต่อ
